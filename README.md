# Alt Recon

A Bash  reconnaissance automation tool.

## Features

- WHOIS lookup
- DNS enumeration
- Passive subdomain discovery (Amass)
- DNS brute force (Gobuster)
- TLS analysis (testssl.sh)

## Usage

Default target:

```bash
./alt-recon.sh
```

Custom target:

```bash
./alt-recon.sh google.com
```
